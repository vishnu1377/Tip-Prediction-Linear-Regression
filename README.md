# Tip-Prediction-Linear-Regression
A Machine Learning project that predicts restaurant tip amounts using Linear Regression. The project includes data preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation using R² score, and an interactive Streamlit web application for real-time predictions.

# Restaurant Tip Prediction using Linear Regression

## Project Overview

This project predicts restaurant tip amounts using Machine Learning techniques. A Linear Regression model was trained on the Seaborn Tips dataset and deployed through a Streamlit web application.

## Features

* Data Analysis using Pandas
* Data Visualization using Seaborn and Matplotlib
* Data Preprocessing
* Train-Test Split
* Multiple Linear Regression
* R² Score Evaluation
* Actual vs Predicted Comparison
* Streamlit Web Application

## Dataset

Source: Seaborn Tips Dataset

Columns:

* total_bill
* tip
* sex
* smoker
* day
* time
* size

## Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Scikit-Learn
* Streamlit

## Model Performance

R² Score: 0.48

pip install -r requirements.txt
